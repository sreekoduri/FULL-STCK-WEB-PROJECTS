🔗 Anchor Element (<a>):The anchor tag is used to create links in HTML.

🔹 Basic Syntax:

<a href="URL">Link Text</a>



href = where to go

Link Text = what user clicks



<a href="https://www.google.com">Go to Google</a>



🖼️ Image Element (<img>):The <img> tag is used to display images on a webpage.

🔹 Basic Syntax:

<img src="image.jpg" alt="Description">



src = image path (REQUIRED)

alt = text if image not visible (IMPORTANT)

<img src="photo.jpg" alt="My Photo">
